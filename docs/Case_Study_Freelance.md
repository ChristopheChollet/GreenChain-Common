# GreenChain Common - Case study freelance

## Resume

GreenChain Common est un projet flagship web3 orienté climate/energy qui unifie quatre briques distinctes dans une seule dApp : vault, market, registry et gouvernance.

## Probleme

Les projets web3 climate/energy sont souvent presentes sous forme de modules separes, ce qui rend difficile la demonstration d'un parcours utilisateur complet et la lecture de KPI consolides.

## Solution

GreenChain Common reunit plusieurs modules au sein d'une interface unique :

- un vault pour le depot et le retrait de capital,
- un market pour la creation et le matching de slots de flexibilite,
- un registry pour l'emission et le retrait de REC,
- une gouvernance pour la creation de propositions et le vote.

Le dashboard centralise egalement des KPI on-chain et une activite recente afin de rendre le projet plus lisible et demonstrable.

Le flow produit peut se resumer simplement ainsi :

`Vault -> Market -> Registry -> DAO`

Cette structure permet a un recruteur ou a un client de comprendre rapidement la logique du projet sans devoir parcourir quatre applications separees.

## Ce que ce projet prouve

- Integration full-stack web3 de plusieurs smart contracts dans une seule application.
- Gestion d'une UX transactionnelle simple avec wallet, writes, retours utilisateur et statuts de transaction.
- Lecture et agregation d'events on-chain pour alimenter un dashboard.
- Capacite a transformer plusieurs MVP separes en une experience produit coherente.

## Valeur pour un client

Ce projet montre la capacite a livrer une dApp MVP demonstrable, avec une architecture modulaire, une interface claire et un parcours utilisateur lisible. Il illustre un niveau de competence utile pour des missions de type :

- integration frontend web3,
- MVP dApp end-to-end,
- dashboard on-chain / KPI,
- UX transactionnelle sur flows blockchain.

## Limites actuelles

- Environnement principalement local / demo.
- Utilisation de mocks sur certains flux.
- Logique volontairement MVP sur plusieurs modules.
- Pas encore d'infrastructure de production complete.
